# Bộ test nhanh logic gợi ý nhóm thuốc

Sau khi chạy `py backend/app.py`, nhập các mô tả sau ở giao diện Trang chủ và bấm **Gợi ý nhóm thuốc**.

| Mô tả test | Nhóm mong đợi |
|---|---|
| Sốt cao, ho, ho có đờm, đau ngực, khó thở | thuốc giãn phế quản hoặc thuốc long đờm / giảm ho |
| Khó thở, khò khè, tức ngực | thuốc giãn phế quản |
| Đau bụng, tiêu chảy, nôn ói | bù dịch và điện giải hoặc thuốc chống nôn |
| Táo bón nhiều ngày, bụng chướng | thuốc nhuận tràng |
| Ngứa da, nổi mẩn đỏ | thuốc kháng histamin |
| Đau đầu, chóng mặt | thuốc giảm đau hạ sốt |
| Đau khớp, sưng khớp | thuốc kháng viêm không steroid |
| Tăng huyết áp, đau đầu | thuốc tim mạch/huyết áp |
| Tiểu buốt, tiểu rắt | thuốc kháng sinh |

Lưu ý: các nhóm kê đơn như kháng sinh, tim mạch/huyết áp vẫn phải hiển thị cảnh báo không tự dùng thuốc.
